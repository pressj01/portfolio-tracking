How it works

  • Each profile stores its own set of holdings, payouts, and history.
  • The Owner profile imports data from the Excel file via the Dashboard import button.
  • Other profiles use the Upload / Edit page to load a CSV or enter holdings manually.
  • All charts, calendar, and analysis pages automatically show data for the active profile.
  • Deleting a profile permanently removes all its data from the database.
